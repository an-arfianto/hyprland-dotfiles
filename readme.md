# Personal Hyprland Dotfiles

- Distro • EndeavourOS
- Window Manager • Hyprland
- Lockscreen • hyplock
- Shell • zsh with Starship + ohmyzsh
- Terminal • foot
- Panel • Waybar
- Notify Daemon • Dunst
- Launcher • Rofi
- File Manager • yazi + nemo
- Text Editor • Neovim with LazyVim
- Colorschemes • Rose Pine + Rose Pine Dawn
