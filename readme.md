# Personal Hyprland Dotfiles

- Window Manager • Hyprland
- Shell • zsh with Starship + ohmyzsh
- Terminal • Kitty
- Panel • Waybar
- Notify Daemon • Dunst
- Launcher • Rofi
- TUI File Manager • yazi 
- GUI File Manager • pcmanfm-qt
- Text Editor • Visual Studio Code + Neovim with LazyVim
